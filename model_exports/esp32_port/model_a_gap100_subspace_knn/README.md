# Model A: Controlled-Gap Subspace KNN

Source artifacts are copied into this directory. `portable_model.json` contains the bagging/KNN parameters, selected feature subsets, training prototypes, class labels, voting rule, metrics, provenance, and ESP32 implementation notes.

This model is selected only because it reproduces 100% on the 8 controlled-gap samples. It is not the 38-sample LOOCV robustness choice.
